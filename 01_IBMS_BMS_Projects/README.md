# 🏢 Integrated Building Management System (IBMS) – Commercial Project

## 📍 Project Overview
Execution and commissioning of an Integrated Building Management System (IBMS) for a commercial/industrial facility, enabling centralized monitoring and control of HVAC and associated systems.

The project focused on improving operational efficiency, system visibility, and real-time monitoring of building services.

---

## 🎯 Objectives
- Centralized monitoring of HVAC systems  
- Integration of field devices with BMS controllers  
- Ensure reliable communication between subsystems  
- Achieve efficient and automated building operations  

---

## 🔧 Scope of Work
- Installation and termination of DDC panels  
- Integration of sensors and actuators (temperature, pressure, flow)  
- HVAC system monitoring and control through BMS  
- Point-to-point wiring verification  
- BMS graphics and system configuration support  
- Testing, commissioning, and functional validation  

---

## ⚙️ Systems & Components Used
- DDC (Direct Digital Controllers)  
- Field Devices: Sensors, Actuators, Control Valves  
- BMS Software Interface (Monitoring & Control)  
- Networking Components (IP-based communication)  

---

## 🔗 System Integration
- Integration with HVAC systems for real-time monitoring  
- Coordination with electrical and mechanical teams  
- Data communication between controllers and central BMS workstation  

---

## 🚧 Challenges & Solutions

### 🔸 Challenge 1: Communication Failure Between Devices  
**Issue:** Intermittent communication loss between field controllers and BMS workstation  
**Solution:**  
- Verified IP addressing and network configuration  
- Checked wiring integrity and controller settings  
- Restored stable communication  

---

### 🔸 Challenge 2: Incorrect Sensor Readings  
**Issue:** Fluctuating temperature values affecting system performance  
**Solution:**  
- Calibrated sensors  
- Verified installation positioning  
- Ensured accurate data transmission  

---

### 🔸 Challenge 3: HVAC Integration Delays  
**Issue:** Coordination gaps between HVAC and BMS systems  
**Solution:**  
- Worked closely with HVAC team  
- Aligned control logic and system points  
- Completed integration successfully  

---

## 📊 Testing & Commissioning

- Performed functional testing of HVAC systems  
- Ensured system performance as per project requirements
